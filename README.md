TO-DO Lista
===============================================


Instação :
---------------------------
* Clone o repositorio ou faça downloado do arquivo Zip.
* Crie um banco de dados vazio.
* Edite o arquivo  conecta_db.php e altere host, user, password e database name.
* Execute o arquivo install.php para a instalação.


Organização:

Index.php: Responsavel por exibir e alterar as informações na lista.
Action.php: Executa as ações e retorna para index.
